# hidolly

> Super powerful and super tiny util to deep cloning of objects and arrays.

## API

### hidolly(obj)
Returns a deep copied of a given object or array.

- `obj`: [Object] - A given object to clone.

```js
var foo = {
  'baz': 'qux'
};

var bar = hidolly(foo);
```
